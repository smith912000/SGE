# SGE Tabs

Page-level views in `src/tabs/`. Each tab corresponds to a section in the app.

---

## Core Astrology

| Tab | Description |
|-----|-------------|
| **NatalTab** | Main natal chart: wheel, planet table, houses. Primary view after compute. |
| **WheelTab** | Multi-mode wheel viewer: Western, Sidereal, Chinese, calendar wheels. |
| **AspectsTab** | Planetary aspects explanation and visualization. |
| **HarmonicsTab** | Harmonic chart analysis (2nd–12th harmonics). |
| **DeepTab** | Deep chart analysis: shadows (Rising, Venus, Mars, Mercury), outer planets (Jupiter, Saturn), pair insights. |
| **StrugglesTab** | Chart challenges: missing elements, hard aspects, shadow placements. |

---

## Traditional Systems

| Tab | Description |
|-----|-------------|
| **ChineseTab** | Chinese astrology: overview, Wuxing, trines, Zi Wei Doushu, generals. |
| **NumerologyTab** | Numerology: Life Path, Expression, Soul Urge, Birthday, and other core numbers. |
| **GrammatologyTab** | Writing systems, letter analysis, Kangxi radicals, cross-script exploration. |
| **CalendarTab** | Šambraielic calendar explorer with date navigation. |

---

## Timing & Returns

| Tab | Description |
|-----|-------------|
| **SolarTab** | Solar return chart analysis. |
| **TransitsTab** | Current transits vs natal chart. |
| **ProgressionsTab** | Secondary progressions (1 day = 1 year). |
| **SynastryTab** | Relationship chart comparison (Person A vs Person B). |
